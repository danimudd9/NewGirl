# New Girl Fan Site

A fan site for the show New Girl, featuring character profile pages and a quote generator.

## Features

- Custom profile pages for all six main characters: Jess, Nick, Schmidt, Winston, Cece, and Coach
- Quote-cycling feature displaying curated quotes for each character
- Built with vanilla HTML, CSS, and JavaScript

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

```bash
git clone https://github.com/danimudd9/NewGirl.git
cd YOUR-REPO-NAME
```

Open index.html in your browser. No build tools or dependencies required.
